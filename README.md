# LM Studio 文本校对工具

使用本地 LM Studio 大语言模型对 Markdown 文件进行智能校对的桌面工具。

## 功能特性

- 📁 支持选择单个.md文件或整个文件夹（批量处理）
- 🔗 检测 LM Studio 连接状态
- 🤖 获取并选择可用模型列表
- 📝 可自定义校对提示词模板
- 📊 实时显示处理日志
- 🎮 显示 GPU 信息（型号、显存使用情况）
- 💾 支持自定义保存位置（默认保存到源文件同目录）

## 依赖要求

- Python 3.8+
- tkinter (Python 内置)
- requests >= 2.28.0
- GPUtil >= 1.4.0

## 安装步骤

### 1. 安装 Python 依赖

```bash
pip install -r requirements.txt
```

### 2. 安装 GPUtil (用于显示GPU信息)

```bash
pip install GPUtil
```

如果安装 GPUtil 失败，可以跳过（程序仍可正常运行，只是无法显示GPU信息）。

## 使用方法

### 1. 启动 LM Studio

确保你的 LM Studio 已启动并加载了模型。

### 2. 运行校对工具

```bash
python proofreader.py
```

### 3. 配置和使用

1. **连接 LM Studio**：
   - 确认 LM Studio 的 API 地址（默认：`http://127.0.0.1:1234/v1`）
   - 点击「检测连接」按钮
   - 选择要使用的模型

2. **选择文件**：
   - 点击「选择文件」选择单个 Markdown 文件
   - 或点击「选择文件夹」批量处理文件夹中的所有 .md 文件

3. **设置保存位置**：
   - 默认保存到源文件相同目录
   - 取消勾选「保存到源文件相同目录」可指定保存位置
   - 输出文件名会自动添加「-已校对」后缀

4. **自定义提示词**（可选）：
   - 根据需要修改校对提示词模板
   - 点击「恢复默认模板」可还原默认设置

5. **开始校对**：
   - 点击「开始校对」按钮
   - 在日志区域查看处理进度

## 故障排除

### 连接失败
- 确保 LM Studio 已启动
- 检查 API 地址是否正确
- 确认 LM Studio 中已启用 API 功能

### 模型列表为空
- 在 LM Studio 中下载并加载模型
- 点击「刷新模型」按钮

### GPU 信息无法显示
- 安装 GPUtil：`pip install GPUtil`
- 或跳过此功能，不影响核心功能
